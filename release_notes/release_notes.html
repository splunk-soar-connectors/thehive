TheHive Release Notes - Published by Splunk June 02, 2022

Version 2.2.0 - Released June 02, 2022
  • Added below mentioned new actions:
    • create alert [PAPP-25493]
    • get alert [PAPP-25498]
    • create ttp [PAPP-25503]
    • list alerts [PAPP-25925]
  • Added below mentioned parameters for the 'create observable' action [PAPP-25508]
    • ticket_type
    • sighted
    • ignore_similarity
  • Fixed a response handling issue for 'create observable' action [PAPP-25474]
Version 2.1.5 - Released January 20, 2022
  • Marked the app as FIPS Compliant [PAPP-21697]
Version 2.1.0 - Released October 4, 2021
  • Added new action 'create task log' to add log entry in the task
  • Added range query to fetch all the observables from TheHive server for the 'get observables' action [PAPP-20230]
  • Updated custom view and output data paths for the 'get observables' action
Version 2.0.4 - Released January 27, 2021
  • Compatibility changes for Python 3 support
